A window with buttons that represent the neck and fretboard of guitar pops up. There are 6 rows for each string, the numbers are the fret numbers. Click the buttons to print the string/fret to the tab. The tab shows up in the console. 

Hold shift while clicking buttons to tab simultaneous notes. Pressing shift also adds a space. Press backspace to backspace tab. Press space to add space on tab.

to run: 
make sure all files in same directory
in terminal, navigate to that directory
type "javac GuitarMain.java"
type "java GuitarMain"
