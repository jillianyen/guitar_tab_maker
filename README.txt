a window with buttons that represent the neck and fret board of guitar pops up. there are 6 rows for each string, the numbers are the fret numbers.
click the buttons to print the string/fret to the tab. the tab shows up in the console. hold shift while clicking buttons to tab simultaneous notes.
press backspace to backspace tab. press space to add space on tab.

to run: 
make sure all files in same directory
in terminal, navigate to that directory
type "javac GuitarMain.java"
type "java GuitarMain"
